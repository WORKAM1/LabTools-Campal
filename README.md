# LabTrack · Suite Analítica

**Propiedad de William Martin Campal**

Aplicación única `index.html` con tres módulos integrados, lista para GitHub Pages.

## Módulos

- **⚗ Diluciones** — Material de vidrio / Tubos 10 mL / Microdiluciones
- **🔥 Fire Assay Pro** — Registro Doré, resultados, procedimiento, informe PDF
- **📈 Calibración ICP/AAS** — Curva de calibración graviométrica, mono y multielementos, informe PDF

## GitHub Pages

1. Subir todos los archivos al repositorio manteniendo la estructura
2. Settings → Pages → Deploy from branch → main → / (root)
3. Acceder a: `https://tuusuario.github.io/LabTools-Campal/`
